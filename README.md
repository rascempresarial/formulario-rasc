# Formulário RASC - Recrutamento de Parceiros

Formulário minimalista para recrutamento de parceiros (Designers, Social Media Specialists e Desenvolvedores) para a agência digital RASC.

## 📋 Sobre

Este projeto é um formulário simples e direto para capturar informações de possíveis colaboradores da RASC. O design é minimalista com paleta de cores: dark, branco e verde.

## 🎨 Design

- **Paleta de Cores**: Dark (#1A1A1A), Branco (#FFFFFF), Verde (#22C55E)
- **Tipografia**: Poppins (Headlines) + Inter (Body)
- **Layout**: Responsivo (2 colunas em desktop, 1 coluna em mobile)
- **Sem dependências externas**: Tudo em um único arquivo HTML

## 📝 Campos do Formulário

- Nome Completo *
- Área de Atuação * (Design, Social Media, Desenvolvimento, Outro)
- Tempo na Área * (Menos de 1 ano, 1-3 anos, 3-5 anos, 5-10 anos, Mais de 10 anos)
- Você trabalha atualmente? * (Sim/Não)
- É home office? (Condicional - aparece se responder "Sim")

## 🚀 Como Usar

1. Abra o arquivo `index.html` em um navegador
2. Preencha o formulário com suas informações
3. Clique em "Enviar Inscrição"

## 💻 Hospedagem

Este arquivo pode ser hospedado em:
- GitHub Pages
- Netlify
- Vercel
- Qualquer servidor web estático

### GitHub Pages

1. Faça push deste repositório para o GitHub
2. Vá para Settings → Pages
3. Selecione "Deploy from a branch" e escolha `main` (ou `master`)
4. Seu site estará disponível em `https://seu-usuario.github.io/formulario-rasc/`

## 🔧 Customização

Para customizar o formulário, edite o arquivo `index.html`:

- **Cores**: Procure por `--rasc-dark`, `--rasc-green`, `--rasc-white` no CSS
- **Textos**: Procure pelos textos em português no HTML
- **Campos**: Adicione novos campos na seção `<form>`
